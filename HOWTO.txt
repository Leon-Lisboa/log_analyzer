### HOWTO.txt (Português)
# Como Usar o Analisador de Logs

1. Execute o seguinte comando para instalar as dependências:
   ```bash
   pip install -r requirements.txt
   ```
2. Execute o programa:
   ```bash
   python log_analyzer.py
   ```
3. Utilize a interface para abrir arquivos de log, filtrar erros e gerar gráficos.
